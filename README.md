# Discord Say dc自动聊天机器人

## 目前应用频道@SaharaLabsAI，怎么玩？

### 1、进去``config``配置目录

#### 1.1 ``channels.json``是dc频道的id

#### 1.2 ``messages.txt``是聊天内容，自己去GPTAI生成

#### 1.3  ``tokens_proxies.txt``是dc的token和代理，格式如下
```txt
token,proxy
```
